# Publia Skills

Agent skills for Publia — publish anything you make to a live, shareable link.

## Install

**Any assistant (Claude Code, Codex, Cursor, …)** — the universal skills installer:

```bash
npx skills add getpublia/skills
```

**Claude Code plugin marketplace** — install it as a first-class plugin:

```bash
/plugin marketplace add getpublia/skills
/plugin install publia@getpublia
```

See [install.md](install.md) for the full autonomous installation flow.
